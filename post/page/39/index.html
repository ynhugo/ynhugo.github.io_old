soul自动闭麦

soul自动闭麦 使用场景 soul开派对，如果是一晚上不关派对且麦克风没有闭麦的情况下，当到了早上的时候，闹钟会响，然后会影响到派对里的人，这个时候有一个可以自动把soul的麦克风关闭的脚本在指定时间点自动运行，避免闹钟透过麦克风影响到其他人。 python脚本 运行环境：termux 脚本运行前提：需要配置好uiautomator2和adb无线调试，无线调试开启后就可以使用127.0.0.1这个ip了。 import sys import uiautomator2 as u2 d = u2.connect('127.0.0.1') # 首先检查当前是否是开麦状态（假设开麦状态下按钮可能有特定属性） # 这里需要根据实际UI情况调整判断条件 if d(resourceId="cn.soulapp.android:id/ivMicButton", description="闭麦按钮").exists: d(resourceId="cn.soulapp.android:id/ivMicButton", description="闭麦按钮").click() crontab定时任务 每天早上7点40自动运行soul自动闭麦python脚本。 40 07 * * * bash $(python ~/soul自动闭麦.py)

多文件字符串查找替换

多文件字符串查找替换 find_str=$1 # 对应当前路径下多文件中要查找的字符串 replace_str=$2 # 对应当前路径下多文件中要替换的新字符串 mapfile -t files < <(find ./ -name "*.c" -o -name "*.h" -o -name "*.cpp" | xargs grep -ainw "${find_str}" | awk -F ':' '{print $1}') # -name指定了要查找的文件类型 # echo "${files[@]}" # 输出所有包含要查找的字符串的文件 for ((i = 0; i < "${#files[@]}"; i++)); do sed -i "s/${find_str}/${replace_str}/g" "${files[${i}]}" # 进行字符串替换，用法为：sed -i "s/新字符串/旧字符串/g file"，由于涉及到变量，因此使用双引号而不是单引号 done 使用示例 ~/find-replace.sh "text1" "text2"
