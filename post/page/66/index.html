termux下的proot-distro安装的Linux路径

termux下的proot-distro安装的Linux路径 /data/data/com.termux/files/usr/var/lib/proot-distro/installed-rootfs/ 即 $PREFIX/var/lib/proot-distro/installed-rootfs/ 使用proot-distro安装linux系统时可以看到这个路径，验证完proot-distro安装linux路径位置后，如果不需要linux可以到linux系统路径下将对应的linux系统删除即可。 proot-distro使用教程 获取linux系统列表 proot-distro list 安装linux系统 例如安装ubuntu或debian proot-distro install ubuntu 或 proot-distro install debian 删除linux系统 例如安装ubuntu或debian proot-distro remove ubuntu 或 proot-distro remove debian

使用Termux在安卓手机上搭建本地Git服务器

使用Termux在安卓手机上搭建本地Git服务器 在局域网使用，需要连接Wifi或开热点 手机正式创建Git的服务仓库 git init --bare test.git –bare：表示创建的是裸仓库，我将该裸仓库创建在家目录下了，对应路径为 ~/test.git 如何使用创建的Git服务仓库 首先，获取Termux的用户名和IP地址，IP地址是IPV4格式的 获取用户名 whoami # 执行该命令后会看到用户名被打印出来 获取ip地址 ifconfig # 执行该命令后可以看到IP地址 然后就是使用Git服务仓库了 例如在Termux中通过 proot-distro 安装的Linux系统，可以进入Termux安装的Linux系统中后可以按照下面的命令格式clone仓库 git clone ssh://username@ipv4:8022/路径 例如 git clone ssh://u0_a223@192.168.3.39:8022/~/test.git 也可以在PC端上按照这个命令格式clone仓库 注意事项 Git服务仓库本身是不能执行push或pull操作的，需要另外建立一个空目录clone Git服务仓库，然后在clone下的仓库中执行操作后再进行push或pull操作。 参考链接 使用Termux在安卓手机上搭建本地Git服务器
