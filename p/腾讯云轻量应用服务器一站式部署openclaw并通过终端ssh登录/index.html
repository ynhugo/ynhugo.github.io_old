腾讯云轻量应用服务器一站式部署OpenClaw并通过终端ssh登录

轻量应用服务器 4核4G3M

活动价79元/月

选择镜像Moltbot

本地生成自定义名称的密钥对 

1
2
3
4
5

# 如果不想使用现有的id_rsa秘钥对，可以生成自定义秘钥对
ssh -t rsa -b 4096 -f ~/.ssh/moltbot

# 如果不想指定秘钥名称，默认id_rsa秘钥对使用最方便
ssh -t rsa -b 4096

配置SSH公钥 

1
2

# 查看公钥
cat ~/.ssh/moltbot

将公钥粘贴到服务器的~/.ssh/authorized_keys中

本地SSH登录 

1
2
3
4
5

# 如果是生成的自定义私钥对，需要指定私钥
ssh -i ~/.ssh/moltbot root@101.43.216.165

# 如果使用id_rsa秘钥对，可以不用指定私钥
ssh root@101.43.216.165

如果不想指定私钥，可以把~/.ssh/id_rsa.pub配置到服务器的authorized_keys中，这样ssh会默认使用私钥id_rsa登录

配置moltbot 

1
2

# 配置moltbot
clawdbot onboard

准备工作 

1
2

# 端口转发
ssh -N -L 18789:127.0.0.1:18789 moltbot

然后在本地访问18789端口即可，如：http://localhost:18789/
