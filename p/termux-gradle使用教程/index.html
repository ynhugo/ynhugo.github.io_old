termux-gradle使用教程

Nov 01, 2024
阅读时长: 4 分钟

安装 Termux 和必要工具

只是创建、编译、运行项目，没有打包生成apk的内容，不过将在网上找到的链接添加到参考链接中了，感兴趣可以看一看。

安装 Termux

如果还没有安装 Termux，可以从 F-Droid 或 Google Play 商店安装。

更新和安装必需的软件

打开 Termux，执行以下命令以更新包管理器和安装 Java 和 Gradle：

1
2
3

  pkg update && pkg upgrade
  pkg install openjdk-17
  pkg install gradle

创建一个简单的 Gradle 项目

实现过程

除了java版本需要指定一下，其他全部默认即可。

查看java版本 

1
2
3
4

创建项目目录 

1
2

  mkdir MyGradleApp
  cd MyGradleApp

初始化 Gradle 项目

使用 Gradle 初始化项目

1
2
3

  gradle init
  # 或
  gradle init --type java-application

构建项目

在项目根目录下运行以下命令以构建项目

1

  gradle build

运行项目

构建成功后，可以运行你的应用

1

  gradle run

完整执行过程

先确认 java 版本，然后执行 gradle init ，过程中一路回车，直到需要输入 java 版本时根据之前查看的版本进行输入，然后接着一路回车到底，最后执行 gradle build 进行编译，再执行 gradle run 运行程序。
