windows下cygiwn安装opencode

操作步骤

在windows下的cygwin下，根据opencode官方文档进行opencode的安装：

1
2
3
4
5

# 如果因网络问题安装失败，可配置代理，根据自己科学上网工具配置的端口进行设置
export https_proxy="http://127.0.0.1:10808"

# 安装opencode
curl -fsSL https://opencode.ai/install | bash

运行opencode 

1
2

# 运行opencode
opencode

使用opencode

默认是免费的模型，如果要使用付费模型，需要付费，我是使用的招商银行的visa信用卡进行的付费，第一次使用付费模型时会提示opencode的付费链接，访问链接后根据提示充值就可以了，每个模型的收费情况不一样，谨慎使用

1
2
3
4
5
6

# 执行后选择其他付费模型
/models

# 如果要选择其他模型提供者，例如Deepseek
/connect
# 然后选择Deepseek，接着粘入从Deepseeek获取到的API Key

临时使用 

1

opencode run "description"
