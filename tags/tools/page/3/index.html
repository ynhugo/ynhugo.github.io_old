telegra

如果想随时随地发表自由化、匿名化的文章，可以使用 telegra.ph，telegram 的产品，因此使用的时候需要科学上网，该网站对中文的支持一般，但是也不影响使用，写第一个中文的时候，再敲击键盘字母的时候会多一个字母出现在文章开头，解决这样 bug 的方式是可以在开头敲一个空格，然后开始正文，写完文章后再将文章开头的空格删除掉即可，写完的文章，点击 PUBLISH 后，复制当前页面链接即可将文章分享给别人，别人不具备编辑权限。 参考链接：https://sspai.com/post/36378
