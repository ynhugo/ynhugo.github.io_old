Android远程控制手机-scrcpy手机版

scrcpy 是一款通过adb远程控制手机的软件，官方有 windows 版本，也有第三方的 mac版。 软件利用 adb 技术，启动一个服务端，实时获取手机画面并通过网络转发到客户端，也就是 windows 或者mac上，目前没有特别实用的 Android 版。 如果有旧手机，那么可以通过内网穿透，将 adb 默认的接入端口 5555 映射到公网，即可将旧手机打造成云手机，随时挂机、操作等。 即便是打开 adb 端口 5555 ，一般情况下，也只能在局域网内使用 ip 访问，如果具备 nas 、或者frp、内网穿透等经验，则可以将手机的调试端口映射到外网，只要映射到外网，则可以随时远程控制 如果家庭网络有 ipv6、或者运营商提供 ipv4，则只需要在路由器打开网络防火墙，供外部访问即可直接远程控制 点击下载 Scrcpy For Android 百度网盘链接：https://pan.baidu.com/s/1vDozKDXevaavt0NM3aHtKg?pwd=turc 参考链接 Android远程控制手机，scrcpy手机版 Scrcpy for Android Android远程控制手机，scrcpy手机版 开机自动执行 shell 的 apk
